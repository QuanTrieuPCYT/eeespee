# eeespee
This repository stores my ESPHome device configurations, in case I lost it again like the last time 😂\
Most of these are devices made by manufacturers, and had ESPHome flashed as a custom firmware.\
**NOTE:** These files are not plug-and-play, you will need a `secrets.yaml` file containing the requested fields.

# Current Devices
- [Desk Lamp](desk-lamp/README.md) - a Mi Desk Lamp 1S
- [Wall Switch](wall-switch/README.md) - Tuya 2CH Wall Switch
- [Decoration Lights](decoration-lights/README.md) - Tuya 10A Relay Switch
